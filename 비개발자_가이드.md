## 📱 방법 1: 가장 쉬운 방법 (추천!)

### ⚡ 무료 배포 서비스 이용하기

개발 환경 없이 바로 웹에서 사용할 수 있는 방법입니다.

#### 1단계: Vercel 무료 배포 (5분)

**Vercel**은 Next.js를 만든 회사의 무료 호스팅 서비스입니다.

1. **Vercel 가입**
   - https://vercel.com 접속
   - "Sign Up" 클릭
   - GitHub 계정으로 가입 (없으면 GitHub도 먼저 가입)

2. **프로젝트 업로드**
   - "Add New..." → "Project" 클릭
   - "Import Git Repository" 대신 **"Deploy from Template"** 선택
   - 아래 단계를 따라하세요:

3. **GitHub에 코드 올리기**
   - https://github.com 가입 (무료)
   - "New repository" 클릭
   - Repository name: `sulloseum-kr` 입력
   - "Create repository" 클릭
   - 다운로드받은 `Sulloseum_KR.zip` 압축 해제
   - 모든 파일을 GitHub에 업로드 (드래그 앤 드롭 가능!)

4. **Vercel에서 배포**
   - Vercel에서 방금 만든 GitHub 저장소 선택
   - "Import" 클릭
   - 완료!

## 🖥️ 방법 2: 컴퓨터에 직접 설치하기

코딩은 몰라도 프로그램 설치는 할 수 있으시죠? 그럼 가능합니다!

### 준비물 (모두 무료!)

✅ 컴퓨터 (Windows, Mac 모두 가능)
✅ 인터넷 연결
✅ 1시간 정도의 시간

## 📝 1단계: 필요한 프로그램 설치 (20분)

### A. Node.js 설치 (필수)

Node.js는 JavaScript를 실행하는 프로그램입니다.

1. https://nodejs.org 접속
2. 큰 녹색 버튼 "**LTS**" (권장) 다운로드 클릭
3. 다운로드된 파일 실행
4. 계속 "다음" 버튼 클릭 (기본 설정 그대로 OK)
5. 설치 완료!

**확인하기:**
- Windows: `시작` → `cmd` 입력 → 명령 프롬프트 실행
- Mac: `Spotlight(⌘+Space)` → `terminal` 입력

다음 명령어 입력:
```bash
node --version
```
버전 번호가 나오면 성공! (예: v20.11.0)

---

### B. Visual Studio Code 설치 (선택, 하지만 권장)

코드를 볼 수 있는 텍스트 에디터입니다.

1. https://code.visualstudio.com 접속
2. "Download" 클릭
3. 설치 파일 실행
4. 계속 "다음" 클릭
5. 완료!

---

## 🗄️ 2단계: 데이터베이스 만들기 (15분)

### Supabase 무료 가입

Supabase는 무료로 데이터베이스를 제공합니다.

1. **가입하기**
   - https://supabase.com 접속
   - "Start your project" 클릭
   - GitHub 계정으로 가입

2. **새 프로젝트 만들기**
   - "New project" 클릭
   - 다음 정보 입력:
     - Name: `sulloseum` (아무거나 괜찮아요)
     - Database Password: 비밀번호 설정 (꼭 메모하세요!)
     - Region: `Northeast Asia (Seoul)` 선택
   - "Create new project" 클릭
   - 2-3분 대기 (프로젝트 생성 중...)

3. **데이터베이스 설정**
   - 왼쪽 메뉴에서 "SQL Editor" 클릭
   - "New query" 클릭
   - 다운로드받은 폴더에서 `supabase-schema.sql` 파일 열기
   - 모든 내용을 복사해서 SQL Editor에 붙여넣기
   - "Run" (또는 F5) 클릭
   - 성공 메시지 확인!

4. **API 키 복사**
   - 왼쪽 메뉴에서 "Settings" (톱니바퀴) 클릭
   - "API" 클릭
   - 다음 두 개를 복사해서 메모장에 저장:
     - `Project URL` (예: https://abcdefgh.supabase.co)
     - `anon public` 키 (긴 문자열)

---

## 🤖 3단계: OpenAI API 키 받기 (10분)

AI가 대화를 생성하려면 OpenAI API 키가 필요합니다.

1. **가입하기**
   - https://platform.openai.com 접속
   - "Sign up" 클릭
   - 이메일 or Google 계정으로 가입

2. **결제 수단 등록** (필수, 하지만 거의 무료!)
   - 상단 메뉴 "Settings" → "Billing" 클릭
   - "Add payment method" 클릭
   - 신용카드 정보 입력
   - 💡 **중요**: GPT-4o-mini는 매우 저렴합니다!
     - 배틀 1회 = 약 10원
     - 처음에 $5 정도만 충전하면 500번 사용 가능

3. **사용 제한 설정** (안전장치)
   - "Usage limits" 탭 클릭
   - "Hard limit": $5 입력 (이상 사용 시 자동 중단)

4. **API 키 생성**
   - 왼쪽 메뉴 "API keys" 클릭
   - "Create new secret key" 클릭
   - 이름: `sulloseum` 입력
   - "Create secret key" 클릭
   - 🚨 **중요**: 나온 키를 복사해서 안전한 곳에 저장! 
     (다시 볼 수 없어요!)

## 💻 4단계: 프로젝트 실행하기 (15분)

### A. 압축 파일 풀기

1. 다운로드받은 `Sulloseum_KR.zip` 파일 찾기
2. 우클릭 → "압축 풀기" (또는 "Extract")
3. 원하는 폴더에 압축 해제 (예: `내문서/Sulloseum_KR`)

---

### B. 환경 변수 설정

1. **파일 이름 바꾸기**
   - 압축 푼 폴더에서 `.env.example` 파일 찾기
   - 이름을 `.env.local`로 변경
   - (Windows에서 파일 확장자가 안 보이면: 폴더 옵션 → 보기 → "알려진 파일 형식의 확장자 숨기기" 체크 해제)

2. **파일 편집**
   - `.env.local` 파일을 메모장(또는 VS Code)으로 열기
   - 아까 복사한 정보 입력:

```env
NEXT_PUBLIC_SUPABASE_URL=https://abcdefgh.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=여기에_supabase_anon_키_붙여넣기
OPENAI_API_KEY=sk-여기에_openai_키_붙여넣기
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

3. 저장하고 닫기

---

### C. 터미널에서 실행

**Windows 사용자:**
1. 압축 푼 `Sulloseum_KR` 폴더로 이동
2. 폴더 안에서 `Shift` + `우클릭`
3. "여기서 PowerShell 창 열기" (또는 "명령 프롬프트 열기") 클릭

**Mac 사용자:**
1. Spotlight(⌘+Space) → `terminal` 입력
2. 터미널에서 다음 입력:
```bash
cd ~/Downloads/Sulloseum_KR
```

**모든 사용자: 다음 명령어를 순서대로 입력**

```bash
# 1. 필요한 패키지 설치 (최초 1회만, 2-3분 소요)
npm install

# 2. 개발 서버 실행
npm run dev
```

잠시 기다리면...

```
✓ Ready in 2.5s
○ Local:   http://localhost:3000
```

이런 메시지가 나옵니다! 🎉

## ❓ 자주 묻는 질문 (FAQ)

### Q1: "npm을 찾을 수 없습니다" 에러가 나요
**A**: Node.js가 제대로 설치되지 않았습니다.
- Node.js를 다시 설치하세요
- 컴퓨터를 재시작하세요

---

### Q2: 포트 3000이 이미 사용 중이라고 나와요
**A**: 다른 프로그램이 3000번 포트를 사용 중입니다.
```bash
# 다른 포트로 실행
npm run dev -- -p 3001
```
그리고 브라우저에서 `http://localhost:3001` 접속

---

### Q3: Supabase 연결이 안 돼요
**A**: 확인사항:
- `.env.local` 파일에 URL과 키가 제대로 입력되었나요?
- Supabase에서 SQL 스크립트를 실행했나요?
- 인터넷이 연결되어 있나요?

---

### Q4: OpenAI API 에러가 나요
**A**: 확인사항:
- API 키가 제대로 입력되었나요? (`sk-`로 시작)
- OpenAI 계정에 결제 수단이 등록되었나요?
- 사용 한도를 초과하지 않았나요?

---

### Q5: 비용이 얼마나 나올까요?
**A**: 
- Supabase: 무료 (Free tier)
- OpenAI: 배틀 1회 약 10-20원
- 한 달에 100번 배틀해도 1,000-2,000원 정도!
- Vercel/Netlify: 무료 (Free tier)

---

### Q6: 친구에게 공유하고 싶어요!
**A**: 두 가지 방법:
1. **로컬 실행 중**: 같은 Wi-Fi 연결 필요
2. **온라인 배포**: Vercel/Netlify 사용 (방법 1 참고)

---

### Q7: 코드를 수정하고 싶어요
**A**: 
- VS Code로 파일 열기
- 수정하고 저장
- 브라우저가 자동으로 새로고침됩니다!
