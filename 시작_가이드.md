## 📚 가이드 종류

### 1️⃣ 비개발자 가이드 (가장 추천!) ⭐
**이런 분들에게 추천:**
- 코딩을 전혀 모르는 분
- 프로그래밍이 처음인 분
- 천천히 따라하고 싶은 분

**파일:** `비개발자_가이드.md`

**특징:**
- ✅ 모든 단계를 자세히 설명
- ✅ 스크린샷 대신 글로 설명
- ✅ 문제 해결 방법 포함
- ✅ 예상 시간: 1시간

---

### 2️⃣ 비주얼 가이드 (그림으로 보고 싶다면)
**이런 분들에게 추천:**
- 글보다 그림이 편한 분
- 빠르게 훑어보고 싶은 분
- 체크리스트로 진행하고 싶은 분

**파일:** `비주얼_가이드.md`

**특징:**
- ✅ 단계별 체크리스트
- ✅ ASCII 아트로 화면 설명
- ✅ 플로우차트 형식
- ✅ 예상 시간: 50분

---

### 3️⃣ 자동 실행 스크립트 (가장 빠름!) ⚡
**이런 분들에게 추천:**
- 빠르게 실행하고 싶은 분
- 최소한의 수동 작업만 하고 싶은 분
- Node.js가 이미 설치되어 있는 분

**파일:**
- Windows: `실행하기-Windows.bat`
- Mac/Linux: `실행하기-Mac-Linux.sh`

**특징:**
- ✅ 클릭 한 번으로 자동 설치
- ✅ 환경 변수 자동 생성
- ✅ 패키지 자동 설치
- ✅ 예상 시간: 10분

**사용 방법:**
1. 파일 더블클릭
2. 화면 지시 따라하기
3. 끝!

---

### 4️⃣ 개발자 가이드 (경험자용)
**이런 분들에게 추천:**
- 개발 경험이 있는 분
- Next.js를 알고 계신 분
- 빠르게 시작하고 싶은 분

**파일:** `README.md`

**특징:**
- ✅ 기술적 설명
- ✅ 간결한 명령어
- ✅ 프로젝트 구조 설명
- ✅ 예상 시간: 15분

## 🚀 빠른 시작 (초간단 버전)

### 필요한 것만 딱 3가지!

1. **Node.js 설치**
   - https://nodejs.org 접속
   - LTS 버전 다운로드
   - 설치

2. **API 키 3개 준비**
   - Supabase URL + Key (supabase.com)
   - OpenAI API Key (platform.openai.com)

3. **실행**
   ```bash
   # Windows: 실행하기-Windows.bat 더블클릭
   # Mac/Linux: 실행하기-Mac-Linux.sh 더블클릭
   ```

끝! 🎉

```

```
