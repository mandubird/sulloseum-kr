## 🚀 각 단계별 상세 가이드

### 1️⃣ Node.js 설치

```
1. 웹브라우저 열기
   ↓
2. 주소창에 입력: nodejs.org
   ↓
3. 이런 화면이 나옵니다:
   ┌────────────────────────────┐
   │  Node.js                   │
   │  [20.11.0 LTS] [다운로드]  │ ← 이거 클릭!
   └────────────────────────────┘
   ↓
4. 파일 다운로드 완료
   ↓
5. 파일 더블클릭 → "다음" → "다음" → "완료"
   ↓
6. ✅ 완료!
```

**확인하는 방법:**
```
Windows: 
시작 메뉴 → "cmd" 입력 → Enter
명령 프롬프트에 입력: node --version

Mac:
Spotlight (⌘ + Space) → "terminal" 입력
터미널에 입력: node --version

v20.11.0 같은 숫자가 나오면 성공! ✅
```

### 2️⃣ Supabase 회원가입 및 설정

**A. 회원가입**
```
1. 브라우저에서 supabase.com 접속
   ↓
2. [Start your project] 버튼 클릭
   ↓
3. [Continue with GitHub] 선택
   (GitHub 계정 없으면 먼저 github.com 가입)
   ↓
4. ✅ 로그인 완료!
```

**B. 프로젝트 생성**
```
1. [New project] 버튼 클릭
   ↓
2. 정보 입력:
   ┌─────────────────────────────┐
   │ Name: sulloseum            │ ← 아무거나 OK
   │ Password: ●●●●●●●●         │ ← 메모하세요!
   │ Region: Seoul              │ ← 한국 선택
   └─────────────────────────────┘
   ↓
3. [Create new project] 클릭
   ↓
4. 2-3분 대기... ☕
   ↓
5. ✅ 프로젝트 생성 완료!
```

**C. 데이터베이스 설정**
```
1. 왼쪽 메뉴에서 찾기:
   [🗄️ SQL Editor] ← 이거 클릭
   ↓
2. [+ New query] 버튼 클릭
   ↓
3. 다운받은 폴더에서 파일 찾기:
   📄 supabase-schema.sql
   ↓
4. 파일을 메모장으로 열기
   → 전체 선택 (Ctrl+A)
   → 복사 (Ctrl+C)
   ↓
5. SQL Editor에 붙여넣기 (Ctrl+V)
   ↓
6. [Run] 버튼 클릭 (또는 F5)
   ↓
7. 화면 아래에 "Success" 메시지 확인
   ↓
8. ✅ DB 설정 완료!
```

**D. API 키 복사 (중요!)**
```
1. 왼쪽 메뉴 맨 아래:
   [⚙️ Settings] 클릭
   ↓
2. 왼쪽 서브메뉴:
   [API] 클릭
   ↓
3. 두 가지를 복사해서 메모장에 저장:
   
   ┌─────────────────────────────────┐
   │ Project URL:                    │
   │ https://abcdefgh.supabase.co   │ ← 복사!
   │                                 │
   │ Project API keys:               │
   │ anon public                     │
   │ eyJhbGc... (긴 문자열)          │ ← 복사!
   └─────────────────────────────────┘
   ↓
4. ✅ API 키 확보!
```

### 3️⃣ OpenAI API 키 받기

**A. 회원가입**
```
1. platform.openai.com 접속
   ↓
2. [Sign up] 클릭
   ↓
3. Google 계정으로 가입 (또는 이메일)
   ↓
4. ✅ 가입 완료!
```

**B. 결제 수단 등록**
```
1. 상단 메뉴에서 찾기:
   [Settings] 클릭
   ↓
2. 왼쪽 메뉴:
   [Billing] 클릭
   ↓
3. [Add payment method] 클릭
   ↓
4. 신용카드 정보 입력
   ↓
5. [Add $5.00] 버튼 클릭
   (처음엔 5달러면 충분! 약 7,000원)
   ↓
6. ✅ 결제 수단 등록 완료!
```

**💡 비용 걱정 마세요!**
```
- 배틀 1회 = 약 10-20원
- $5 = 약 250-500회 배틀 가능!
- 한도 설정으로 초과 방지 가능
```

**C. 사용 한도 설정 (안전장치)**
```
1. [Billing] 페이지에서:
   [Usage limits] 탭 클릭
   ↓
2. 설정:
   Hard limit: $5 입력
   (이 금액 도달 시 자동 중단)
   ↓
3. ✅ 안전장치 설정 완료!
```

**D. API 키 생성**
```
1. 왼쪽 메뉴에서:
   [🔑 API keys] 클릭
   ↓
2. [+ Create new secret key] 클릭
   ↓
3. 이름 입력:
   Name: sulloseum
   ↓
4. [Create secret key] 클릭
   ↓
5. 나온 키를 복사! (한 번만 보여줍니다!)
   
   ┌─────────────────────────────┐
   │ sk-proj-abcd1234...        │ ← 복사하고 저장!
   └─────────────────────────────┘
   ↓
6. ✅ API 키 확보!
```

### 4️⃣ 프로젝트 설정 및 실행

**A. 압축 해제**
```
1. Sulloseum_KR.zip 파일 찾기
   ↓
2. 우클릭 → [압축 풀기]
   ↓
3. 원하는 위치에 풀기
   (예: 내문서/Sulloseum_KR)
   ↓
4. ✅ 압축 해제 완료!
```

**B. 환경 변수 설정**
```
1. 압축 푼 폴더 열기
   ↓
2. 파일 찾기:
   📄 .env.example
   ↓
3. 파일 이름 바꾸기:
   .env.example → .env.local
   ↓
4. .env.local 파일을 메모장으로 열기
   ↓
5. 아까 복사한 정보 입력:
   
   ┌────────────────────────────────────┐
   │ NEXT_PUBLIC_SUPABASE_URL=          │
   │ https://abcdefgh.supabase.co      │ ← Supabase URL
   │                                    │
   │ NEXT_PUBLIC_SUPABASE_ANON_KEY=     │
   │ eyJhbGc...                         │ ← Supabase 키
   │                                    │
   │ OPENAI_API_KEY=                    │
   │ sk-proj-abcd...                    │ ← OpenAI 키
   │                                    │
   │ NEXT_PUBLIC_APP_URL=               │
   │ http://localhost:3000              │ ← 그대로 두기
   └────────────────────────────────────┘
   ↓
6. 저장하고 닫기 (Ctrl+S)
   ↓
7. ✅ 환경 변수 설정 완료!
```

**C. 쉬운 실행 방법 (스크립트 사용)**

**Windows 사용자:**
```

   ↓
2. 더블클릭!
   ↓
3. 검은 화면이 나타나면서 자동으로:

   - 서버 실행
   ↓
4. "✓ Ready" 메시지가 나오면 성공!
   ↓
5. ✅ 실행 완료!
```

**Mac/Linux 사용자:**
```
1. 압축 푼 폴더에서 파일 찾기:
   📄 실행하기-Mac-Linux.sh
   ↓
2. 우클릭 → [열기] 선택
   (또는 터미널에서: ./실행하기-Mac-Linux.sh)
   ↓
3. 자동으로 설치 및 실행!
   ↓
4. ✅ 실행 완료!
```

## ❓ 문제 해결

### "npm을 찾을 수 없습니다"
```
→ Node.js가 설치 안 됨
→ 해결: Node.js 다시 설치
→ 컴퓨터 재시작
```

### "포트 3000이 사용 중"
```
→ 다른 프로그램이 사용 중
→ 해결: npm run dev -- -p 3001
→ 브라우저: localhost:3001
```

### "환경 변수 에러"
```
→ .env.local 파일 확인
→ API 키가 제대로 입력되었나?
→ 띄어쓰기, 줄바꿈 확인
```

### "Supabase 연결 안 됨"
```
→ SQL 스크립트 실행했나?
→ API 키가 맞나?
→ 인터넷 연결 확인
```
